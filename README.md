## 📌 Sobre o projeto

Este é o meu portfólio pessoal, desenvolvido com o objetivo de apresentar minhas habilidades tanto como Analista de Dados quanto como Desenvolvedor Front-End.

Aqui compartilho projetos práticos voltados para análise de dados, criação de dashboards e também desenvolvimento de interfaces web, unindo análise e visualização com uma boa experiência do usuário.
